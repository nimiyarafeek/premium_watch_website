# DAXIAOYU Premium Mechanical Watch Landing Page

A premium, fully responsive, and animated product landing page for the **DAXIAOYU Fully Automatic Mechanical Watch**. Built with semantic HTML, modern CSS styling (including glassmorphism and custom gradients), and vanilla JavaScript.

## 🔗 Links

* **Live Demo:** [https://nimiyarafeek.github.io/premium_watch_website/](https://nimiyarafeek.github.io/premium_watch_website/)
* **GitHub Repository:** [https://github.com/nimiyarafeek/premium_watch_website](https://github.com/nimiyarafeek/premium_watch_website)

---

## ✨ Features

* **Hollow Skeleton Aesthetics:** Sleek dark-mode theme featuring gold gradient highlights, neon glow backdrops, and modern typography.
* **Frictionless Interactive Checkout:** A slide-out checkout drawer with real-time state transitions (checkout form to order success panel) powered by Vanilla JS.
* **100% Responsive Design:** Smooth fluid layouts engineered using CSS Grid and Flexbox for mobile, tablet, and desktop viewports.
* **Sticky Header Navigation:** Interactive navbar that transitions opacity and shadow upon scrolling down.

---

## 🛠️ Technology Stack

* **Structure:** Semantic HTML5
* **Design & Animations:** Vanilla CSS3 (Custom properties, transitions, and keyframes)
* **Interactions:** Vanilla JavaScript (ES6+)
